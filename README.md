# Operating System in Development (Phase 2)
This is a basic operating system that can read the jobs and give output where we implemented the paging concept and simulated how paging works for OS.
You can get all the information and instructions about this project in the file named **AOSPROJECT2.pdf**
Till now the progress of our os is it can read the data from job, load it in memory and on demand give output. 
In furthur versions there will be addition of more advancements. 

This project is made with IntelliJ Idea so preferably you can directly open this as a project in IntelliJ Idea.
